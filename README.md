🚀 MelliADM (formerly Tele2Rub)

انتقال خودکار فایل از تلگرام به روبیکا و دانلودر همه‌کاره شبکه‌های اجتماعی — سریع، ساده، بدون هزینه

پشتیبانی از ارسال فایل تا ۲ گیگابایت

🧠 معرفی

MelliADM
یک ابزار قدرتمند است که علاوه بر دریافت فایل‌ها از تلگرام و ارسال به سیو مسیج (Saved Messages) روبیکا، می‌تواند از سایت‌ها، لینک‌های مستقیم و شبکه‌های اجتماعی (یوتیوب، اینستاگرام، توییتر و...) مستقیماً فایل را دریافت کرده، به کاربر در تلگرام تحویل داده و در صف آپلود روبیکا قرار دهد.

✨ قابلیت‌ها

📥 فوروارد انواع فایل از تلگرام به روبیکا

🎥 جدید: دانلودر داخلی از اینستاگرام، یوتیوب، توییتر و... (بدون نیاز به ربات ثانویه) با تعیین کیفیت /mdl

🌐 جدید: دانلود کامل سورس و قالب یک سایت /webpage

🔗 دانلود از لینک‌های اینترنتی مستقیم /link

🧾 ارسال متن‌ها به صورت فایل txt.

🛡 حالت Safe Mode برای ارسال فایل‌ها به صورت ZIP رمزدار

⚡سیستم صف برای جلوگیری از کرش

⏱ محدودیت زمان آپلود قابل تنظیم

🛠 نصب سریع

ابتدا پروژه را دریافت کنید:

git clone [https://github.com/caffeinexz/Tele2Rub.git](https://github.com/caffeinexz/Tele2Rub.git)
cd Tele2Rub


نصب وابستگی‌ها (اگر لینوکس هستید نصب ffmpeg برای دانلودرهای یوتیوب الزامی است):

sudo apt update
sudo apt install ffmpeg -y
pip install -r requirements.txt


اجرای پروژه:

python3 main.py


⚙️ تنظیمات

یک فایل .env در ریشه پروژه بسازید:

API_ID=عدد_API
API_HASH=کد_API
BOT_TOKEN=توکن_ربات
RUBIKA_SESSION=rubsession


📥 نحوه استفاده و دستورات ربات

ارسال فایل: کافیست هر فایلی را در ربات تلگرام آپلود یا فوروارد کنید.

/mdl : ورود به بخش مدیا دانلودر (یوتیوب/اینستاگرام/...)

/webpage : دانلود قالب کامل سایت به صورت فایل زیپ

/link : ورود به بخش دانلود لینک مستقیم

/safemode on/off : روشن/خاموش کردن ارسال به صورت فایل زیپ

/caption on/off : وضعیت ارسال کپشن فایل‌ها

/del : حذف شناسه از صف

/delall : پاکسازی کامل صف
